# 개발환경설정

## 1. 설치 유의사항

1. Wi-Fi 설정
* 설치할때 Wi-Fi를 한국(KR)이 아닌 미국(US)을 잡아야 한다.
  + 이미 KR로 지정했다면 
    - 시작 -> Preferences -> Raspberry Pi Configuation -> Localisation -> WiFi Country를 US로 변경
    - 또는 Terminal 에서 sudo raspi-config 실행 -> Localisation Option -> I4 Change Wi-Fi Country 변경
